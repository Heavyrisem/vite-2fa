# Vite BoilerPlate

## Description

this is a boilerplate for vite projects

- Vite
- ESlint
- Prettier
- Tailwind
- Twin.Macro
- Docker
- Docker-Compose

## Installation

```bash
$ git clone
$ yarn install
```

## Usage

### For Development

```bash
$ yarn dev
```

### For Production

```bash
$ docker-compose up -d
```
